## 工具箱
- `sub-convert.py` - 订阅解析/过滤/格式转换
- `rules.yaml` - Clash分流规则（广告屏蔽/国内直连/流媒体）
- `adblock.txt` - 广告+追踪器域名黑名单
- `direct.txt` - 国内直连域名列表
